# CubewiseBit官网文档仓库

使用[Jekyll](http://jekyllcn.com/)构建，[GitHub Pages](https://docs.github.com/cn/pages/setting-up-a-github-pages-site-with-jekyll/about-github-pages-and-jekyll)支持

> 运行如下命令，在本地预览

```shell
# clone项目
git clone https://github.com/CubewiseBit/cubewisebit.github.io.git
cd cubewisebit.github.io
# 安装依赖
bundle install
# 编译并运行
bundle exec jekyll serve -l
# 打开浏览器预览
open http://127.0.0.1:4000
```
